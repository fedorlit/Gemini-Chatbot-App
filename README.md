# Gemini Chatbot App 🤖

App de chat para Android que utiliza **Google Gemini** para ofrecer respuestas multimodal (texto + imágenes) usando el modelo 2.5 flash lite

### 🛠️ Tecnologías usadas
- **Kotlin**, **Compose**, **Gemini SDK**, **Coil**, **ViewModel & Flow**.

### 🚀 API
-API generada desde la página oficial de Google AI Studio.
