# Gemini Chatbot App 🤖

App de chat para Android que utiliza **Google Gemini** para ofrecer respuestas multimodal (texto + imágenes).

### 🛠️ Tecnologías usadas
- **Kotlin**, **Compose**, **Gemini SDK**, **Coil**, **ViewModel & Flow**.

### 🚀 API
- API sacada de la página oficial de Google AI Studio
