# Gemini Chatbot App 🤖

App de chat para Android que utiliza **Google Gemini** para ofrecer respuestas inteligentes y análisis multimodal (texto + imágenes).

### ✨ Características
- **IA Multimodal**: Análisis de texto e imágenes en tiempo real.
- **UI Moderna**: Desarrollada con **Jetpack Compose** y **Material 3**.
- **Dark Mode**: Interfaz azulada optimizada para modo oscuro.
- **Gestión de Medios**: Selección y deselección ágil de imágenes.

### 🛠️ Tecnologías
- **Kotlin**, **Compose**, **Gemini SDK**, **Coil**, **ViewModel & Flow**.

### 🚀 Configuración
1. Clonar repo.
2. Añadir API KEY de [Google AI Studio](https://aistudio.google.com/).
